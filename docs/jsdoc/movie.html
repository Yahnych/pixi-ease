Name Type Default Description
wait number 0 optional n milliseconds before starting animation (can also be used to pause animation for a length of time)
pause boolean optional start the animation paused
repeat boolean | number optional true: repeat animation forever n: repeat animation n times
reverse boolean | number optional true: reverse animation (if combined with repeat, then pulse) n: reverse animation n times
continue boolean | number optional true: continue animation with new starting values n: continue animation n times
load function optional loads an animation using a .save() object note the * parameters below cannot be loaded and must be re-set
